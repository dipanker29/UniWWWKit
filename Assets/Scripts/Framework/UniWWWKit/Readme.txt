UniWWWKit V0.1

This framework useful establish a communiation with server using REST APIs.
Have a support for both POST & GET APIs.

Use WWWConfig.cs to configure server url and few other properties.

This framework make use of Newtonsoft JSON parsing framework to parse the JSON data.
You can use your own JSON parser or custom code in any classes.
